# Meeting Room Booking API

This project is a simple meeting room booking API built with Node.js and TypeScript using an in-memory data store.  
It supports creating, cancelling, and listing room reservations per room while enforcing basic business rules.

## How to run the project

Install dependencies:

```
npm install
```

Build the project:

```
npm run build
```

Start the server:

```
npm start
```
